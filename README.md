Columbus Weather Dashboard

This is a simple weather webpage that shows the forecast for the Columbus, Ohio area.

It gets all its data from the National Weather Service (NWS) public API.

Features

Shows current weather and 7-day forecast for 36 zip codes.

"Refresh All" button updates the data for all locations.

"Export CSV" button lets you download all the weather data in a spreadsheet.

Works on both desktop and mobile phones.

How to Use

Download the columbus-weather-nws-csv.html file.

Double-click the file to open it in any web browser.
